This project is a La\+TeX to Markdown converter, designed as part of the Software Lab course at IIT Delhi. The converter takes a La\+TeX file as input and outputs a Markdown file with properly formatted content. The converter handles common La\+TeX structures such as sections, subsections, bold, italics, code blocks, tables, and lists.\hypertarget{md_readme_autotoc_md6}{}\doxysection{Features}\label{md_readme_autotoc_md6}

\begin{DoxyItemize}
\item Convert La\+TeX sections, subsections, and subsubsections to Markdown headings.
\item Translate La\+TeX bold and italic text to Markdown bold and italics.
\item Process La\+TeX {\ttfamily verbatim} environments into Markdown code blocks.
\item Handle La\+TeX tables and lists (ordered and unordered).
\item Convert La\+TeX images into Markdown image syntax.
\end{DoxyItemize}\hypertarget{md_readme_autotoc_md7}{}\doxysection{Project Structure}\label{md_readme_autotoc_md7}
